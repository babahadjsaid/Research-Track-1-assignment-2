duplicate the data of "/odom" topic

Parameters
[in]PublishFreqDefine the frequency of publishing.

Subscribes to:
° /odom ° /cmd_vel

Publishes to:
° duplicatedOdom

Description :

This node subscribes to the '/odom' and '/cmd_vel' topics and publishes a custom message containing the pose, linear velocity, and angular velocity to the 'duplicatedOdom' topic. The frequency of publishing is determined by the 'PublishFreq' parameter.
